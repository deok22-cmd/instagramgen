## ▶ 다음 세션 이어가는 법

새 PC면 클론 후 venv 재생성, 같은 PC면 그대로:

```powershell
cd "D:\0. LAMP\Instagram Gen"        # 또는 git clone 후 그 폴더
.\.venv\Scripts\python.exe -m insta_gen formats   # 동작 확인
$env:PYTHONPATH = "src"; $env:PYTHONUTF8 = "1"
```

## ⚠️ 보안 메모 — 꼭 확인

채팅에 평문으로 노출된 Gemini 키(`AIza…KmOg`)는 **Google AI Studio에서 폐기(rotate) 권장**. 어떤 파일·repo에도 저장되지 않았음을 확인했지만, 노출 자체가 위험이므로 폐기가 안전.

새 키 사용은 환경변수로만:

```powershell
$env:GEMINI_API_KEY = "<새 키>"
```

캐시된 응답이 있는 입력은 키 없이도 재현됨 (`assets/cache/ai/`).

## 📌 메모

- AI 응답 캐시는 (provider, model, system, user) 해시 기반 → 같은 입력 재호출 무료·즉시
- 출력 폴더 구조: `output/<yymmdd>/<keyword>/...svg`
  - `yymmdd` 우선순위: `--date` > 입력경로의 6자리 폴더 > 오늘
- `python -m insta_gen preview` → `output/index.html`에서 모든 SVG 컨택트시트
- 알려진 후속 정리: 8자리 hex color는 P8에서 fill+fill-opacity 분리 검토, CJK 줄바꿈 닫는 구두점 폴리시
